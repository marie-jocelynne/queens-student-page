# queens-student-page

This repository contains my personal student webpage created for **CISC 121 – Homework 2**.  
The goal of this project was to practice using GitHub, GitHub Pages, and basic HTML/CSS.

## Live Website
You can view the deployed site here:  
https://marie-jocelynne.github.io/queens-student-page/

## What’s in this repo
- `index.html` – main webpage content  
- `style.css` – page styling  
- Image file for the profile photo  

## Tools Used
- HTML  
- CSS  
- GitHub & GitHub Pages  

## Notes
This project demonstrates setting up a public repository, deploying a site with GitHub Pages, and using pull requests as part of a peer review workflow.
